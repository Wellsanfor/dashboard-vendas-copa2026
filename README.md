# 🏆 Dashboard de Vendas - Copa do Mundo 2026

Este projeto apresenta um dashboard interativo de vendas focado em produtos temáticos para a Copa do Mundo de 2026. O objetivo foi transformar dados brutos em insights estratégicos para tomada de decisão.

## 📊 O Dashboard
O dashboard foi desenvolvido utilizando uma combinação de **Python** para geração/tratamento de dados e **Excel** para a interface visual e interatividade.

**Principais funcionalidades:**
- **KPIs Dinâmicos:** Acompanhamento em tempo real de Faturamento, Lucro, Volume de Itens e Margem.
- **Análise Regional:** Gráfico de desempenho por região geográfica.
- **Mix de Produtos:** Visualização das categorias mais vendidas.
- **Interatividade:** Filtros (Slicers) por Região e Categoria que atualizam todos os gráficos simultaneamente.

## 🛠️ Tecnologias Utilizadas
- **Python 3.12+**: Utilizado para gerar a base de dados fictícia e estruturar o arquivo Excel.
  - Bibliotecas: `Pandas`, `Numpy`, `Openpyxl`.
- **Microsoft Excel**: Utilizado para o design do Dashboard, Tabelas Dinâmicas e Segmentadores de Dados.
